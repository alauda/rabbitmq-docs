# Обновление патч-версии

С учетом необходимости исправления уязвимостей или выпуска новых функций, будут проводиться обновления патч-версий промежуточного программного обеспечения. В процессе обновления патч-версии Pod’ы будут перезапускаться по одному, что может привести к кратковременному прерыванию обслуживания. Поэтому рекомендуется выбрать подходящее время для обновления. Когда патч-версия доступна для обновления, рекомендуется выполнить обновление как можно скорее, чтобы применить последние исправления к компонентам.

## Настройка стратегии обновления патч-версий

### Процедура

<Tabs>
  <Tab label="Веб-консоль">
    1. Перейдите на страницу деталей экземпляра.

    2. Нажмите на **Политика обновления патч-версии**.

    3. Выберите **Ручное** или **Автоматическое**.

    4. Нажмите на **Обновить**, чтобы завершить настройки.
  </Tab>
</Tabs>

## Обновление по одному

### Предварительные условия

Когда патч-версия доступна, на странице списка экземпляров появится значок обновления, а на странице деталей экземпляра будет показано уведомление об обновлении.

### Процедура

<Tabs>
  <Tab label="Веб-консоль">
    1. Перейдите на страницу деталей экземпляра.

    2. Нажмите **Обновить сейчас** рядом с уведомлением об обновлении.

    3. Ознакомьтесь с уведомлением об обновлении, чтобы понять изменения версий компонентов и плагинов.

    4. Нажмите **Обновить сейчас**.

    5. Дождитесь изменения статуса экземпляра на "Работает", что указывает на успешное обновление.
  </Tab>
</Tabs>

## Массовое обновление

### Предварительные условия

Пользователи должны иметь права администратора платформы.

### Процедура

<Tabs>
  <Tab label="Веб-консоль">
    1. Нажмите на **Управление обновлением** в левом навигационном баре.

    2. Выберите **оператора**, с которым нужно работать, и перейдите на страницу деталей оператора.

    3. Выберите экземпляры, которые вы хотите обновить, и нажмите **Массовое обновление**.

    4. Ознакомьтесь с уведомлением об обновлении, чтобы понять изменения версий компонентов и плагинов.

    5. Нажмите **Обновить сейчас**.

    6. Дождитесь изменения статуса экземпляра на "Работает", что указывает на успешное обновление.
  </Tab>
</Tabs>
