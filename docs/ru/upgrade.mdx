# Обновление

Служба Alauda Streaming для RabbitMQ будет выполнять обновления в зависимости от настроенной стратегии обновления:

- **Автоматическое** : Автообновления запускаются сразу после обнаружения новых версий компонентов.
- **Ручное** : Требуется ручное одобрение перед началом процесса обновления.
