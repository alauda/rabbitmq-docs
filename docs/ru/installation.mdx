# Установка

Этот справочник описывает процесс установки Alauda Streaming Service для RabbitMQ, включая его необходимые зависимости. Следуйте этим шагам последовательно, чтобы обеспечить правильное развертывание.

## Установка основных компонент Alauda Container Platform Data Services

::: tip Дополнительно
Основные компоненты Alauda Container Platform Data Services предоставляют элементы интерфейса управления Web Console. Эта установка может быть пропущена для окружений, использующих только администрирование через CLI.
:::

### Предварительные условия

1. Получите пакет установки основных компонентов Alauda Container Platform Data Services, совместимый с вашей версией платформы.
2. Используйте возможность публикации приложений платформы для публикации основных компонентов Alauda Container Platform Data Services в кластер `global`.

### Процедура

<Tabs>
  <Tab label="Web Console">
    1. В левом навигационном меню перейдите в **Marketplace** > **Cluster Plugins**.
    2. Выберите кластер `global`.
    3. Нажмите кнопку действия рядом с плагином **Alauda Container Platform Data Services Essentials** > **Install**.
  </Tab>
</Tabs>

## Установка фреймворка Alauda Container Platform Data Services RDS

::: tip Дополнительно
Фреймворк Alauda Container Platform Data Services RDS предоставляет уровень управления сервисом API. Эта установка может быть пропущена для окружений, использующих только администрирование через CLI.
:::

### Предварительные условия

1. Получите пакет установки фреймворка Alauda Container Platform Data Services RDS, совместимый с вашей версией платформы.
2. Используйте возможность публикации приложений платформы для публикации фреймворка Alauda Container Platform Data Services RDS в любой кластер, где требуются возможности представления Data Services.

### Процедура

<Tabs>
  <Tab label="Web Console">
    1. Войдите в платформу и перейдите на страницу **Administrator**.
    2. В левом навигационном меню выберите **Marketplace** -> **OperatorHub**, чтобы перейти на страницу **OperatorHub**.
    3. Найдите **Alauda Container Platform Data Services RDS Framework**, нажмите **Install**, и перейдите на страницу развертывания.

    Параметры конфигурации:

    | **Параметр**             | **Рекомендуемая конфигурация**                                                                                                                |
    | :------------------------ | :--------------------------------------------------------------------------------------------------------------------------------------------- |
    | **Канал**                | Канал по умолчанию — **stable**.                                                                                                             |
    | **Версия**               | По умолчанию выбрана последняя версия.                                                                                                       |
    | **Режим установки**       | **Cluster**: Все пространства имен в кластере используют одного Оператора для создания и управления экземплярами, что приводит к меньшему потреблению ресурсов. |
    | **Место установки**       | Выберите **Рекомендуемое пространство имен**: автоматически создается, если оно еще не существует.                                           |
    | **Стратегия обновления**  | **Manual**: При наличии новой версии в OperatorHub требуется ручное подтверждение для обновления Оператора до последней версии.                 |

    4. На всплывающей странице установки используйте настройки по умолчанию, затем нажмите **Install**, чтобы завершить развертывание **Alauda Container Platform Data Services RDS Framework**.
  </Tab>
</Tabs>

## Установка Alauda Streaming Service для RabbitMQ

### Предварительные условия

1. Получите пакет установки, совместимый с вашей версией платформы.
2. Загрузите плагин в кластеры, которые собираются использовать возможности бизнес-компонентов Alauda Streaming Service для RabbitMQ.

### Процедура

<Tabs>
  <Tab label="Web Console">
    1. Войдите в платформу и перейдите на страницу **Administrator**.
    2. В левом навигационном меню выберите **Marketplace** -> **OperatorHub**, чтобы перейти на страницу **OperatorHub**.
    3. Найдите **Alauda Streaming Service для RabbitMQ**, нажмите **Install**, и перейдите на страницу развертывания.

    Параметры конфигурации:

    | **Параметр**             | **Рекомендуемая конфигурация**                                                                                                                |
    | :------------------------ | :--------------------------------------------------------------------------------------------------------------------------------------------- |
    | **Канал**                | Канал по умолчанию выбирается автоматически.                                                                                                   |
    | **Версия**               | По умолчанию выбрана последняя версия.                                                                                                       |
    | **Режим установки**       | **Cluster**: Все пространства имен в кластере используют одного Оператора для создания и управления экземплярами, что приводит к меньшему потреблению ресурсов. |
    | **Место установки**       | Выберите **Рекомендуемое пространство имен**: автоматически создается, если оно еще не существует.                                           |
    | **Стратегия обновления**  | **Manual**: При наличии новой версии в OperatorHub требуется ручное подтверждение для обновления Оператора до последней версии.                 |

    4. Проверьте параметры конфигурации в диалоговом окне установки и нажмите **Install**, чтобы развернуть Alauda Streaming Service для RabbitMQ.
  </Tab>
</Tabs>
