# Журнал

Понимание журналов операций экземпляров на уровне контейнера, таких как журналы ошибок Rabbitmq. Правильное использование журналов поможет вам быстро выявлять проблемы и устранять неисправности и исключения.

## Процедура

<Tabs>
  <Tab label="Web Console">
    1. Нажмите на **RabbitMQ** в левом навигационном меню.

    2. Нажмите на ***Название пространства имен***.

    3. Нажмите на ***Название экземпляра***.

    4. На вкладке **Realtime Log** установите критерии поиска.

       - Чтобы найти журналы для конкретных Pods, в опции **Pod** выберите Pod.

    5. Подождите немного и просмотрите результаты.

    6. (По желанию) Нажмите <span style={{display: 'inline-block'}}>![](../../en/assets/44buttonexport.png)</span> для экспорта журналов.
  </Tab>
</Tabs>
