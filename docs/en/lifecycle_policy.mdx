## Release Policy

A new minor version of the Alauda Streaming Service for RabbitMQ is released approximately every few months.

The Alauda Streaming Service for RabbitMQ follows a **rolling support model**:

- Each version remains supported **until the next version is officially released**.
- There is no fixed end date for support; the duration depends on the release cadence.
