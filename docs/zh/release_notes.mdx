# 发布说明

## v4.0.0

### 新增和优化的功能

#### 组件-平台解耦

灵雀云的RabbitMQ流服务的生命周期与平台的解耦，这意味着当客户需要某个功能或修复某个bug时，RabbitMQ团队可以立即发布，而无需等待平台的共同发布。

#### 组件详情页的实时日志

RabbitMQ详情页上的日志已被实时日志替代，实时更新，方便故障排查和问题分析。

#### 新版本支持

灵雀云的RabbitMQ流服务现在支持RabbitMQ服务器v3.12.14。
