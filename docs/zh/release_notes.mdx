# 发布说明

## v4.0.0

### 新增和优化功能

#### 组件-平台解耦

灵雀云的 RabbitMQ 流服务的生命周期已与平台的生命周期解耦，这意味着当客户需要某个功能或修复 bug 时，RabbitMQ 团队可以立即发布，而无需等到平台版本的发布。

#### 组件详情页的实时日志

RabbitMQ 详情页上的日志已被实时日志替换，实时更新，方便故障排查和问题分析。

#### 新版本支持

灵雀云 RabbitMQ 流服务现在支持 RabbitMQ 服务器 v3.12.14。
