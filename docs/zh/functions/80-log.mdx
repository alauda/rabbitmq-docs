# 日志

了解容器层级的实例操作日志，例如 RabbitMQ 错误日志。合理使用日志可以帮助您快速定位问题，处理故障和异常。

## 步骤

<Tabs>
  <Tab label="Web 控制台">
    1. 在左侧导航菜单中点击 **RabbitMQ**。

    2. 点击 ***命名空间名称***。

    3. 点击 ***实例名称***。

    4. 在 **实时日志** 选项卡上，设置搜索条件。

       - 要查找特定 Pods 的日志，在 **Pod** 选项中，选择一个 Pod。

    5. 等待片刻，查看结果。

    6. （可选）点击 <span style={{display: 'inline-block'}}>![](../assets/44buttonexport.png)</span> 导出日志。
  </Tab>
</Tabs>
