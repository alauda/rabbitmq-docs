# 启动与停止实例

一键启动和停止功能满足日常开发和运维需求，使您能够快速暂停或启动中间件实例。在计划停机、节点负载过大以及开发测试环境等场景中，您可能需要暂时释放 CPU 和内存资源，而无需删除实例。一键启动和停止功能帮助您节省资源并灵活管理中间件实例，从而降低成本，提高效率。

## 停止实例

### 前提条件

实例处于 **处理**、**运行**、**异常** 或 **未知** 状态。

### 步骤

<Tabs>
  <Tab label="Web 控制台">
    1. 点击左侧导航栏中的 **RabbitMQ**。

    2. 点击 ***命名空间名称***。

    3. 点击 ***实例名称***。

    4. 点击 **操作** > **停止**，并确认。
  </Tab>
</Tabs>

### 停止后不可用的功能

| 功能                       | 描述                                                       |
| -------------------------- | ---------------------------------------------------------- |
| **操作**                  | 无法重启、更新或修改调度配置。                            |
| **参数配置**              | 无法更新或导出参数。                                      |
| **检查**                  | 当前实例的检查结果将无效。                                |

## 启动实例

### 前提条件

实例处于 **停止中** 或 **已停止** 状态。

### 步骤

<Tabs>
  <Tab label="Web 控制台">
    1. 点击左侧导航栏中的 **RabbitMQ**。

    2. 点击 ***命名空间名称***。

    3. 点击 ***实例名称***。

    4. 点击 **操作** > **启动**，并确认。
  </Tab>
</Tabs>
