# 删除实例

删除实例除了删除创建的 CR 资源，还会删除 RabbitMQ 根据 CR 资源创建的 Statefulset，Service, Configmap，Secret, PersistentVolumeClaim 等资源。

## 操作步骤

<Tabs>

<Tab label="CLI">

如果实例没有开启持久化，或者需要保留实例的持久化数据，执行以下命令来删除实例：

```bash
# delete rabbitmq instance directly
kubectl -n default delete rabbitmqclusters <Instance Name>
```

删除实例默认不会删除创建的 PVC，可使用如下命令删除:

```bash
kubectl -n default delete pvc -l app.kubernetes.io/component=rabbitmq,app.kubernetes.io/part-of=rabbitmq,app.kubernetes.io/name=<Instance Name>
```

</Tab>

<Tab label="Web Console">

1. 在左侧导航栏中，单击 **RabbbitMQ**。

2. 单击 ***命名空间的名称***。

3. 在实例列表中选择要删除的实例，进入实例详情页面。

4. 在**实例详情** 页面的右上角，点击**操作**按钮，展开功能菜单。

5. 点击功能菜单中的**删除**，会弹出删除确认框。

6. 对开启了数据持久化的实例，可勾选“删除持久卷申明”来释放存储，或者不勾选来保留数据。

7. 在弹出框中数据要删除的实例名称

8. 点击**删除**

</Tab>

</Tabs>
