# 删除实例

删除实例不仅会删除创建的 CR 资源，还会删除 StatefulSet、Service、ConfigMap、Secret、PersistentVolumeClaim 以及 RabbitMQ 基于 CR 资源创建的其他资源。

## 步骤

<Tabs>
  <Tab label="CLI">
    如果实例尚未配置持久性，或者需要保留实例的持久数据，请执行以下命令删除实例：

    ```bash
    # 直接删除 rabbitmq 实例
    $ kubectl -n default delete rabbitmqclusters <实例名称>
    ```

    默认情况下，删除实例不会移除创建的 PVC。您可以使用以下命令将其删除：

    ```bash
    $ kubectl -n default delete pvc -l app.kubernetes.io/component=rabbitmq,app.kubernetes.io/part-of=rabbitmq,app.kubernetes.io/name=<实例名称>
    ```
  </Tab>

  <Tab label="Web 控制台">
    1. 在左侧导航栏中，点击 **RabbitMQ**。

    2. 点击 ***命名空间名称***。

    3. 在实例列表中，选择要删除的实例，进入实例详细信息页面。

    4. 在 **实例详细信息** 页面右上角，点击 **操作** 按钮展开功能菜单。

    5. 在功能菜单中点击 **删除**，将弹出删除确认对话框。

    6. 对于启用数据持久性的实例，您可以勾选“删除持久卷声明”以释放存储，或者不勾选以保留数据。

    7. 在弹出对话框中，输入要删除的实例名称。

    8. 点击 **删除**。
  </Tab>
</Tabs>
