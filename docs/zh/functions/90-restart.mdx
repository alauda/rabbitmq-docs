# 重启实例

当实例在运行过程中遇到操作失败、连接数过多或其他未识别问题时，可以通过重启实例来解决这些问题。

## 注意事项

在重启过程中，Pods 将会一个接一个地重启，这可能会导致服务短暂中断。

## 前提条件

实例必须处于 **运行中** 状态。

**备注**：

- 如果实例处于 **处理中** 状态，这表示该实例可能正在正常启动。请耐心等待。

- 如果实例未能启动或出现异常，导致其处于 **处理中** 或 **错误** 状态，则在健康检查超时后会自动重启。请耐心等待。

## 步骤

<Tabs>
  <Tab label="Web 控制台">
    1. 在左侧导航栏中点击 **RabbitMQ**。
    2. 点击 ***命名空间名称***。
    3. 点击 ***实例名称***。
    4. 点击 **操作** > **重启**，并确认。
  </Tab>
</Tabs>
