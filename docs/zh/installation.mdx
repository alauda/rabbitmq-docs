# 安装

本指南概述了 Alauda Streaming Service for RabbitMQ 的安装过程，并包括其所需的依赖项。请按顺序执行以下步骤，以确保正确部署。

## 安装 Alauda Container Platform Data Services Essentials

::: tip 可选
Alauda Container Platform Data Services Essentials 提供了 Web 控制台管理界面的组件。如果您的环境仅使用 CLI 管理，可以跳过此安装步骤。
:::

### 先决条件

1. 下载与您的平台版本兼容的 Alauda Container Platform Data Services Essentials 安装包。
2. 使用平台的应用发布功能将 Alauda Container Platform Data Services Essentials 发布到 `global` 集群。

### 步骤

<Tabs>
  <Tab label="Web 控制台">
    1. 在左侧导航栏中，前往 **市场** > **集群插件**。
    2. 选择 `global` 集群。
    3. 点击 **Alauda Container Platform Data Services Essentials** 插件旁边的操作按钮 > **安装**。
  </Tab>
</Tabs>

## 安装 Alauda Platform Data Services RDS Framework

::: tip 可选
Alauda Platform Data Services RDS Framework 提供了 API 服务管理层。如果您的环境仅使用 CLI 管理，可以跳过此安装步骤。
:::

### 先决条件

1. 下载与您的平台版本兼容的 Alauda Platform Data Services RDS Framework 安装包。
2. 使用平台的应用发布功能将 Alauda Platform Data Services RDS Framework 发布到任何希望使用 Alauda 应用服务 Web 控制台功能的集群中。

### 步骤

<Tabs>
  <Tab label="Web 控制台">
    1. 登录到平台，并进入 **平台管理** 页面。
    2. 在左侧导航栏中，选择 **市场** -> **操作员中心** 以进入 **操作员中心** 页面。
    3. 找到 **Alauda Platform Data Services RDS Framework**，点击 **安装**，进入部署页面。

    配置参数：

    | **参数**                | **推荐配置**                                                                                                                               |
    | :---------------------- | :---------------------------------------------------------------------------------------------------------------------------------------- |
    | **渠道**                | 默认渠道是 **稳定**。                                                                                                                      |
    | **版本**                | 默认选中最新版本。                                                                                                                        |
    | **安装模式**            | **集群**：集群下的所有命名空间共享一个 Operator 来创建和管理实例，降低资源消耗。                                                              |
    | **安装位置**            | 选择 **推荐命名空间**：如果不存在，将自动创建。                                                                                            |
    | **升级策略**            | **手动**：当操作员中心有新版本时，需要手动确认以升级操作员到最新版本。                                                                     |

    4. 在弹出的安装页面，使用默认配置，然后点击 **安装** 完成 **Alauda Platform Data Services RDS Framework** 的部署。
  </Tab>
</Tabs>

## 安装 Alauda Streaming Service for RabbitMQ

### 先决条件

1. 下载与您的平台版本兼容的插件安装包。
2. 将插件上传到计划利用 Alauda Streaming Service for RabbitMQ 业务组件功能的集群中。

### 步骤

<Tabs>
  <Tab label="Web 控制台">
    1. 登录到平台，并进入 **平台管理** 页面。
    2. 在左侧导航栏中，选择 **市场** -> **操作员中心** 以进入 **操作员中心** 页面。
    3. 找到 **Alauda Streaming Service for RabbitMQ**，点击 **安装**，进入部署页面。

    配置参数：

    | **参数**                | **推荐配置**                                                                                                                               |
    | :---------------------- | :---------------------------------------------------------------------------------------------------------------------------------------- |
    | **渠道**                | 默认渠道会自动选择。                                                                                                                       |
    | **版本**                | 默认选中最新版本。                                                                                                                        |
    | **安装模式**            | **集群**：集群下的所有命名空间共享一个 Operator 来创建和管理实例，降低资源消耗。                                                              |
    | **安装位置**            | 选择 **推荐命名空间**：如果不存在，将自动创建。                                                                                            |
    | **升级策略**            | **手动**：当操作员中心有新版本时，需要手动确认以升级操作员到最新版本。                                                                     |

    4. 在弹出的安装页面，使用默认配置，然后点击 **安装** 完成 **Alauda Streaming Service for RabbitMQ** 的部署。
  </Tab>
</Tabs>
